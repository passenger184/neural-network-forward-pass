# Neural Network Forward Pass From Scratch

A beginner-friendly deep learning project implementing a multi-layer neural network forward pass manually using NumPy.

## Features

- Multi-layer neural network
- Manual weight initialization
- ReLU activation function
- Matrix multiplication using NumPy
- Shape tracking through layers

## Network Architecture

Input Layer (5)
→ Hidden Layer 1 (6)
→ Hidden Layer 2 (8)
→ Hidden Layer 3 (8)
→ Output Layer (3)

## Technologies Used

- Python
- NumPy

## Concepts Practiced

- Linear algebra
- Matrix multiplication
- Neural networks
- Forward propagation
- Activation functions
- Weight matrices
- Bias vectors

## How To Run

```bash
python forward_pass.py
```
